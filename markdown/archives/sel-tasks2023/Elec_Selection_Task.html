Electronics Selection Task 2023

Hi Everyone, Welcome to the specialized selection task for electronics.

This Selection Task is for those who previously have some background in electronics and are interested in joining the A.T.O.M society.

This task is going to be offline.

Problem Statement

We will be disclosing it on the day of the Task

The task will involve the following concepts:

  1. Charlieplexing

  2. Multiplexing

  3. Interrupts

  4. PWM

Stuff you will be needing :

  1. Microcontroller (Arduino, ESP32, or NodeMCU )

  2. LEDs ( min 4 )

  3. Breadboard

  4. Bunch of Jumper wires ( min 10 )

  5. Resistors (recommended 220 ohms , min 5 )

  6. Push Button ( min 1 ) or ( or any other form of input peripheral )

Note

You have to bring your own electronic stuff and a laptop.

Time and Venue

Venue: Room no.401A, 4th Block Basement

Date: 16th December 2022

Time: 12:00 PM onwards
