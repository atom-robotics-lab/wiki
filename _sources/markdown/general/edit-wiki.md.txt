# Editing this wiki

## Step 1

* Fork the github repository by clicking on the fork button on the top right

## Step 2

* Clone your fork of the repository 

## Step 3

* Make changes, add and commit
* Install and [setup the mdbook tool](https://github.com/rust-lang/mdBook#installation)
* Build and test if the mdbook is running

## Step 4

* Push to your fork

## Step 5

* Create a merge request to the main repository
