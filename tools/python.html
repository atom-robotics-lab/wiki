What is Python ?

Python is a popular programming language created by Guido van Rossum, and released in 1991. It is a high-level, general-purpose programming language known for its simple syntax and high usability.

Python has widespread applications in :

  • Machine learning/Artificial intelligence
  • Web development (server-side)
  • Software development
  • Robotics
  • Mathematics
  • System scripting

Please refer the Official Python Documentation for more information.

For installation of Python, refer the Official Python Website
